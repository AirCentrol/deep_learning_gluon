一些调参时的体会：

- 数据增强会减小训练准确率和测试准确率之间的差值，收敛也会变慢，不过一般会获得更好的效果。
- 从cifar-10的结果看，模型的改进能获得确实的效果。
- 更强的权重衰减有可能提升模型的精度。
- 过早或者过晚decay学习率都可能造成精度的下降。
